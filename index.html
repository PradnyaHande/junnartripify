About this site

This is a single-file interactive tourism planner for the city of Junnar. It contains sample information for 10 places, category filters, Google Map links, and a simple planning algorithm that assigns time per place.

How to use

  1. Browse places or search by category.
  2. Open Trip Planner and enter your arrival time and how many hours you have.
  3. Click Generate Plan to create an itinerary. Use the map links to navigate.

Developer notes

You can edit the placesData array in the JavaScript section to add real descriptions, exact Google Maps URLs, and high-quality images.
